# Map I/O

Restrict a chosen input to a single display

## Installation 

Install map-io with the bash script `./scripts/install.sh`
    
## Run Locally

map-io can be executed without installing with the bash script `./scripts/dev.sh`
## Usage/Examples

```bash
map-io
```
